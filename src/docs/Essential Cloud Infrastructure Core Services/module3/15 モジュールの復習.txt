このモジュールではGCPが提供するストレージおよびDBサービスを学習しました
Cloud Storageは、フルマネージドのオブジェクトストアです
Cloud SQLは フルマネージドのMySQLおよびPostgreSQLデータベースサービスです
Cloud SpannerはRDBサービスで、トランザクションの整合性、グローバルなスケーリング、高可用性が特徴です
Cloud FirestoreはフルマネージドのNoSQLドキュメントDBです
Cloud BigtableはフルマネージドのNoSQLワイドカラム型DBです
Cloud MemorystoreはRedis対応のフルマネージドインメモリデータストアです
インフラの観点から、各種状況に適したサービスやその使用方法を理解することを目標としました
完全なデータ戦略の策定については、このコースでは触れませんが、GCPでのデータエンジニアリングと機械学習に関するコースで取り上げています
