Cloud Memorystoreの概要を簡単に説明します
Memorystore for Redisはフルマネージドのインメモリデータストアサービスで、Googleが管理するスケーラブルで安全な高可用性インフラで稼働します
拡張性と可用性の高いセキュアなRedis、サービスを使ってGCP上でアプリを実行すると、複雑なRedisデプロイを管理する負担なく高パフォーマンスを実現できます
その分、コードの作成に時間をかけて優れたアプリの構築に注力できます
Cloud Memorystoreは複雑なタスクも自動化し、高可用性、フェイルオーバー、パッチ適用、モニタリングなどを自動的に処理します
高可用性インスタンスは２つのゾーンで複製され、可用性99.9%のSLAを提供します
アプリに必要なミリ秒未満のレイテンシや、スループットを簡単に実現できます
最下位の階層と最小のサイズから開始し、アプリの可用性への影響を最小限に抑えながらインスタンスを容易に拡張できます
Cloud Memorystoreは最大300GBのインスタンスと、１秒あたり12GBのネットワークスループットをサポートします
Cloud Memorystore for RedisはRedisプロトコルに完全に対応しているため、コードを変更しなくてもインポート/エクスポート機能を使って、オープンソースのRedisからMemorystoreにリフト＆シフトできます
新しいツールを習得する必要もなく、クライアントライブラリの既存のすべてのツールを使えます
