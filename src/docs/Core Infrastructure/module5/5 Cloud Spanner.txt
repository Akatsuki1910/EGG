Google Cloudの第３のストレージオプションは、Cloud Spannerです
フルマネージドのリレーショナルデータベースサービスで、水平スケーリング可能で強整合性を備え、SQL対応です
Google固有のミッションクリティカルな、アプリとサービスでテストされたSpannerは、Googleの800億ドルのビジネスを支えています
Cloud Spannerは次のような要件のある用途に向いています
テーブル結合とセカンダリインデックスを使う
SQL RDB管理システム
組み込みの高可用性
高度なグローバル整合性
２テラバイトを超えるデータベース
膨大な数の１秒あたりの入力/出力処理
つまり、１秒あたり数万回以上の読み取りと書き込みがある場合です
