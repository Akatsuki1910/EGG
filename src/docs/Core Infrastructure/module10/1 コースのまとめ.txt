お疲れ様でした
「Google Cloud Fundamentals: Core Infrastructure」コースを修了しました
先に進む前に、学習したことを振り返ってみましょう
モジュール１は、Google Cloudとクラウドコンピューティングの概要で、具体的にはマネージドインフラストラクチャの概念、IaaSとPaaSによるマネージドサービス、Google Cloudネットワーク、インフラストラクチャ全体でのセキュリティの重視、オープンソースライセンスによる主要な技術要素の公開、Google Cloudの料金体系と請求ツールについてです
モジュール２ではGoogle Cloudのリソース階層に注目しました
これは４つのレベルから成り、リソース、プロジェクト、フォルダ、組織ノードです
ポリシーの定義とその下方継承についても、学びました
Cloud Identity and Access Management（Cloud IAM）をいつ使用するか、そしてGoogle Cloudにアクセスする４つの方法、Cloud Console、Cloud SDKとCloud Shell、API、Cloud Consoleモバイルアプリです
モジュール３では、Compute Engineの動作の仕組みとして、仮想マシンと仮想ネットワーキングに注目しました
学んだのは、VPC（Virtual Private Cloud）、Compute Engineの自動スケーリング機能、Google Virtual Private Cloudの重要な互換性機能であるルーティングテーブル、ファイアウォール、VPCピアリング、共有VPCです
これらすべてがネットワーク管理の必要性を軽減します
Cloud Load Balancingについても見ました
全トラフィックに対する完全分散型のソフトウェア定義マネージドサービスです
最後に、オンプレミスや他のクラウドネットワークを、Google VPCとどのように相互接続できるかを比較しました
モジュール４では、Google Cloudの５つの主要ストレージオプションである、Cloud Storage、Cloud Bigtable、Cloud SQL、Cloud Spanner、Firestoreに注目しました
また、Cloud Storageを構成する４つのストレージクラスを見ました
Standard Storageは、頻繁にアクセスするデータに使用され、Nearline StorageとColdline Storageはアクセス頻度の低いデータに使用されます
そして、Archive Storageです
モジュール５ではコンテナについて学びました
これは、コードとその依存関係を含む目に見えない箱です
３つのコンテナベースのプロダクトを紹介しました
Kubernetesはオープンソースプラットフォームで、コンテナ化したワークロードやサービスを管理し、Google Kubernetes Engineは、Googleがホストするクラウド内のマネージドKubernetesサービスです
Anthosは、最新のハイブリッド/マルチクラウド分散型システムやサービス管理に対応します
モジュール６では、クラウドでのアプリケーション開発に注目しました
App Engineはフルマネージドのサーバーレスプラットフォームで、大規模なウェブアプリの開発とホストに利用できます
また、スタンダードとフレキシブルの２つのApp Engine環境、Google Cloudが提供する２つのAPI管理ツール、Cloud EndpointsとApigee Edgeも取り上げました
Cloud Runはマネージドコンピューティングプラットフォームで、ウェブ要求やPub/Subイベントを介してステートレスコンテナを実行します
モジュール７では、クラウドでの開発とデプロイに注目し、Cloud Source Repositoriesについて学びました
Google Cloudでホストされる多機能のGitリポジトリです
Cloud Functionsは軽量のイベントベースの非同期コンピューティングツールで、単一目的の関数を作成できます
Terraformではテンプレートを使用して、構成ファイルを書くのと同じ方法でアプリケーション環境の仕様を記述できます
最後は、Google Cloudでのロギングとモニタリングに注目しました
システムのパフォーマンスと信頼性を測定する４つのゴールデンシグナルは、レイテンシ、トラフィック、飽和度、エラーです
サービスレベル指標（SLI）とサービスレベル目標（SLO）、サービスレベル契約（SLA）はすべて、ゴールデンシグナル指標に対してシステムに設定される目標です
最後に、Googleの統合オブザーバビリティツールである、Cloud MonitoringとCloud Logging Error Reporting、デバッガ、Cloud Trace、Cloud Profilerを紹介しました
このコースではGoogle Cloud導入の基本を学びました
さらなるトレーニングと実践演習については、cloud.google.com/trainingにある各種学習プログラムをご覧ください

Google Cloud認定資格の詳細については、cloud.google.com/certificationsをご覧ください
