Cloud Dataprepについて少し説明します
Cloud Dataprepはインテリジェントデータサービスで、分析、レポート、機械学習に使用する構造化データと非構造化データを、視覚的に探索、クリーニング、準備できます
Cloud Dataprepはサーバーレスで規模に関係なく稼働し、デプロイや管理が必要なインフラはありません
また理想的なデータ変換操作が、UI入力のたびに提案、予測されるため、コードを書く必要がありません
スキーマ、データ型、可能な結合、異常が自動的に検出されるので、データのプロファイリング作業に時間を費やすことなく、データ分析に専念できます
Cloud DataprepはTrifactaが運営する統合パートナーサービスで、業界トップクラスのデータ準備ソリューションであるTrifacta Wranglerを基盤としています
GoogleはTrifactaとの緊密な連携により、シームレスなUXを実現しています
そのため事前のソフトウェアインストールと追加のライセンス費用は不要で、継続的な運用オーバーヘッドも発生しません
Cloud Dataprepはフルマネージドでお客様のデータ準備ニーズに応じて、オンデマンドでスケーリングされるのでお客様は分析に専念できます
これはCloud Dataprepアーキテクチャの例です
ご覧のようにCloud Dataprepを利用して、BigQuery、Cloud Storageまたはアップロードファイルから元データを準備した後、データをCloud Dataflowなどの変換パイプラインに取り込めます
調整されたデータはBigQueryやCloud Storageにエクスポートして、分析や機械学習を行えます
